{% trans "How financing works for property purchases in Turkey" %}

{% trans "Turkey offers bank mortgages and developer installments for both residents and foreign buyers. Terms, documentation, and maximum loan amounts vary by bank, buyer profile, and property valuation. Below is a practical summary to help you prepare." %}

{% trans "Who can apply" %}

  • {% trans "Turkish citizens and residents: broad bank options and terms." %}
  • {% trans "Foreign buyers: mortgages are available at selected banks, subject to valuation and local policy." %}

{% trans "Typical loan terms" %}

  • {% trans "Currency: mostly TRY-denominated mortgages; FX loans are limited." %}
  • {% trans "Tenor: commonly 5–10 years depending on bank and profile." %}
  • {% trans "Loan-to-Value (LTV): determined by the bank’s appraisal; foreigners often qualify for lower LTVs than residents." %}
  • {% trans "Interest: variable by bank and market; request an up-to-date quote." %}

{% trans "Required documents (common)" %}

  • {% trans "Passport/ID and biometric photo" %}
  • {% trans "Turkish tax number (Vergi Numarası)" %}
  • {% trans "Proof of income (salary slips, tax returns, bank statements)" %}
  • {% trans "Address proof (utility bill or residency document)" %}
  • {% trans "Property appraisal report (SPK valuation) arranged via bank" %}

{% trans "Process at a glance" %}

  1. {% trans "Pre-qualification: discuss budget, LTV, and estimated rates with the bank." %}
  2. {% trans "Open a Turkish bank account and obtain a tax number if you don’t have one." %}
  3. {% trans "Submit documents; bank orders independent valuation of the property." %}
  4. {% trans "Final approval: the loan amount is set based on the valuation report." %}
  5. {% trans "Signing and disbursement: earthquake insurance (DASK) is typically required before deed transfer." %}

{% trans "Fees and mandatory items" %}

  • {% trans "Appraisal (valuation) fee" %}
  • {% trans "Loan allocation/processing fee (varies by bank)" %}
  • {% trans "DASK earthquake insurance (mandatory) and property insurance" %}
  • {% trans "Title deed taxes and notary fees (paid at transfer)" %}

{% trans "Money transfers" %}

{% trans "International buyers typically fund down payments and closing via SWIFT transfers in foreign currency. Banks may request source-of-funds documentation. Currency conversion and payment documentation for the title office are handled in coordination with your bank and notary." %}

{% trans "Alternatives to bank mortgages" %}

  • {% trans "Developer installment plans: staged payments during constrinsuction or short-term post-handover schedules." %}
  • {% trans "Personal loans or home equity from your home country (if applicable)." %}
